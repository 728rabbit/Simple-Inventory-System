## Key Features:

Add, edit, and delete inventory items

View current stock levels and values ​​in real-time

Set low-stock alerts to prevent stockouts

Quickly search for items by name, category, or location

Track inbound and outbound inventory records

Export data to Excel / CSV files

Multiple user roles (Admin / Standard User)

## Use Cases:

Daily inventory management for small shops, warehouses, and studios

Management of personal belongings or household supplies

Demonstration project for learning inventory management workflows
